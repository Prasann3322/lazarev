# 🚀 Lazarev Website Clone

A modern, animation-rich website clone inspired by **Lazarev – AI + Product Design Agency**.

This project was built to practice advanced frontend concepts including smooth scrolling, GSAP animations, ScrollTrigger integration, SVG manipulation, and interactive UI components.

---

## 🎯 Project Purpose

This is a **practice clone project** created to:

- Master GSAP animations
- Implement ScrollTrigger with Locomotive Scroll
- Practice SVG hover transitions
- Build advanced interactive layouts
- Improve animation timing and performance

The design concept is inspired by Lazarev, but all code and implementation were written from scratch.

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- GSAP (GreenSock Animation Platform)  
- ScrollTrigger  
- Locomotive Scroll  
- SVG Animations  

---

## ✨ Features Implemented

- Smooth scrolling with Locomotive Scroll  
- Scroll-based animations using GSAP ScrollTrigger  
- Pinned sections & sticky button behavior  
- Animated navigation dropdown  
- Hover-based SVG transitions  
- Rotating SVG elements on scroll  
- Interactive `<details>` dropdown with animated arrow rotation  
- Newspaper-style text wrapping layout  
- Advanced animation timing using cubic-bezier curves
